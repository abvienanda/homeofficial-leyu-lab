# homeofficial-leyu-lab

本仓库用于归档和发布多个独立 HTML 页面，主要作为静态资源存放与展示用途。

## 项目简介

这是一个通用型 HTML 页面归档仓库，不针对任何特定网站或域名。仓库中的页面均为独立编写或整理的静态 HTML 文件，可用于学习、展示或参考。

## 目录说明

```
.
├── index.html          # 仓库首页 / 导航页
├── pages/              # 归档页面存放目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/             # 样式、脚本、图片等资源文件
    ├── css/
    ├── js/
    └── images/
```

- `pages/` 目录下存放主要的 HTML 页面文件
- `assets/` 目录存放页面引用的资源文件

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中打开
- 页面之间无强依赖关系，可单独使用
- 部分页面可能引用 `assets/` 中的公共资源
- 归档内容可能不定期更新或调整

## 维护说明

- 本仓库仅作为静态页面归档使用，不提供动态服务
- 页面内容可能根据需要进行增删或修改
- 如遇到页面显示异常，请检查资源文件路径是否正确
- 欢迎通过 Issues 反馈问题或提出建议

## 使用方式

1. 克隆或下载本仓库到本地
2. 直接在浏览器中打开 `index.html` 或 `pages/` 目录下的任意页面
3. 也可通过 GitHub Pages 等方式在线访问（需自行配置）
